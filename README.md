# TugasBesarAKA
Analisis perbandingan kompleksitas algoritma pencarian nilai maksimum dengan pendekatan iteratif dan rekursif, pada data Ujian Tulis Berbasis Komputer(UTBK). Perbandingan dilakukan untuk melihat perbedaan kinerja kedua metode ketika ukuran data semakin besar, sehingga diketahui pendekatan mana yang lebih efisien dalam mengolah data berskala besar.
